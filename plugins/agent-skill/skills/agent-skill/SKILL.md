# Agent Skill 起手式

教你把一份 SOP 包成 agent 讀得懂的 skill。

## 流程
1. 用一句話寫 description（什麼時候該觸發它）。
2. body 只放編排與步驟，細節丟進 references/（需要範本時讀 references/template.md）。
3. 每個 reference 開頭寫「何時取用」。

## Files

Read these on demand when the workflow above calls for them:

- `references/template.md` — 動手寫第一個 skill 時取用
